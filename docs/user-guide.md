# Руководство пользователя LaTeX Editor ЛЭТИ

## Вход

1. Откройте в браузере адрес сервиса (например https://latex.etu.ru).
2. На экране входа введите логин и пароль (для демо-доступа: логин `demo`, пароль `demo`).
3. Нажмите «Войти».

## Создание проекта из шаблона

1. Нажмите «Новый проект» в шапке или в боковой панели.
2. Введите название проекта.
3. При запросе укажите id шаблона (например `article` или `beamer`) или оставьте пусто для пустого проекта.
4. После создания откроется список файлов; выберите главный .tex-файл для редактирования.

## Редактирование и компиляция

- В боковой панели отображаются проекты и файлы текущего проекта. Выберите проект, затем файл.
- Редактор поддерживает подсветку LaTeX; изменения сохраняются автоматически через 1–2 секунды после ввода.
- Нажмите «Скомпилировать» или используйте Ctrl+Enter / F5 для сборки PDF.
- Включите «Авто-компиляция», чтобы сборка запускалась после каждого сохранения.
- Справа отображается предпросмотр PDF и лог компиляции. При ошибках в логе можно кликнуть по строке с ошибкой для перехода к соответствующей строке в редакторе.

## История версий

- Нажмите кнопку «История» в шапке при открытом файле.
- В модальном окне отобразится список версий. Нажмите «Восстановить» у нужной версии, чтобы подставить её содержимое в редактор.

## Работа с файлами

- Правый клик по файлу в списке файлов открывает контекстное меню: «Новый файл», «Переименовать», «Удалить».
- При создании нового файла укажите путь относительно проекта (например `sections/intro.tex`).

## Горячие клавиши

- Ctrl+S — сохранить текущий файл.
- Ctrl+Enter или F5 — скомпилировать проект.
- Кнопка «Полный экран» над превью открывает PDF в отдельном окне.
